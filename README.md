# Python Workbench

This repository contains several Python scripts created as part of school or personal learning projects. Each script demonstrates basic programming concepts and provides fun utilities or games.

## Included Programs

- `days-calculator.py` - Calculates the number of days between two dates.
- `encrybtion-decrybtion.py` - A simple encryption and decryption tool (note: filename contains a typo).
- `number-scrabble-game.py` - A command-line implementation of a number scrabble game.
